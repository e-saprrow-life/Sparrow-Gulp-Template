=== GULP =========================================================================================
    Комманды текущей сборки: npm run
        d - dev - начать разработку
        b - build - скомпилировать проект
        z - zip - создать архив папки dest
        c - continue - продолжить разработку 
        upld - upload - выгрузить проект на хост в удаленную папку
        clrd - clean remote dir - очистить удаленную папку
        i - install - установить все пакеты

    gulp.series(...) - выполняет задачи последовательно
    gulp.parallel(...) - выполняет задачи паралельно (одновременно)
    (!) gulp-clean-css может не работать если папка исходника имеет символ #. Используй правило наименования переменных.

    (i) К команде можно передавать флаги типа --flag, а функция process.argv.includes('--test') позволит записать 
        куда то в настойки был ли этот флаг вызван при вызове програмы
    
    (i) Установка Gulp: 
        - Установить Node js
        - Power Shell (Admin) - Set-ExecutionPolicy RemoteSigned   (Расширяет права) 
        - Power Shell (Admin) - npm install --global gulp-cli      (Установить Gulp глобально) 
        - Установить gulp в проект npm install gulp
        - установить нужные плагины
